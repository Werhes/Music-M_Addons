# VkMusicTime

**Версия:** 1.0.0
**Автор:** Werhes
**Тип:** Утилита

## Описание

**VkMusicTime** — это аддон для VK M, который отслеживает время, проведённое в приложении. Ведите учёт своего времени и получайте уведомления о достижениях каждый час!

## Возможности

- ⏱ **Общее время** — сколько всего времени вы провели в VK M
- 📅 **Время за сегодня** — сколько времени вы слушаете музыку сегодня
- 🎵 **Время прослушивания** — сколько времени вы реально слушали музыку
- 🏆 **Достижения** — уведомления при достижении каждого часа (1ч, 2ч, 3ч... и далее: 24ч, 50ч, 100ч, 200ч, 500ч, 1000ч)
- 💾 **Автосохранение** — данные сохраняются каждые 30 секунд и при закрытии приложения

## Использование

1. Установите аддон через **Магазин дополнений**
2. Аддон активируется автоматически
3. При первом запуске появится приветственное уведомление
4. Каждый час вы будете получать уведомление с поздравлением и статистикой
5. При достижении особых порогов (24, 50, 100, 200, 500, 1000 часов) — специальные уведомления

## Где хранятся данные

Все данные сохраняются в базу данных VK M (таблица `SettingsTable`) с префиксом `vkmusictime_`:
- `vkmusictime_total_seconds` — общее время в секундах
- `vkmusictime_listening_seconds` — время прослушивания музыки
- `vkmusictime_today_seconds` — время за сегодня
- `vkmusictime_today_date` — дата последнего сброса
- `vkmusictime_last_achievement` — последнее полученное достижение (в часах)
- `vkmusictime_first_launch` — дата первого запуска

## Совместимость

Требуется VK M версии 1.0.0 или выше.
